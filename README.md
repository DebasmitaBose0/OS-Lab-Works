## Assignments 1, 2 and 3

## 📌 Introduction
This repository contains a comprehensive collection of **Shell Programming (Bash) programs** completed as part of the **Linux / UNIX Shell Programming Laboratory**.

The objective of these assignments is to understand and apply:
- Basic Linux commands
- Shell scripting fundamentals
- Conditional statements
- Looping constructs
- Arithmetic and logical operations
- User input and output handling
- Floating-point calculations using `bc`

All programs are written in **Bash shell** and executed in a **Linux / WSL (Ubuntu)** environment.

---

## 🖥️ Software & Environment Requirements
- Operating System: **Linux / Windows with WSL**
- Shell: **Bash**
- Text Editor: `vi` / `vim`
- Utility Package: `bc` (for floating-point calculations)

### Install `bc` (if not already installed):
```sh
sudo apt install bc
```

## 📁 Repository Contents
This repository contains shell programming assignments, Python process and memory management examples, and deadlock detection exercises.

- `22.1.26.txt` — Assignment 1 includes WSL installation steps and shell programs for addition, multiplication, number comparison, and odd/even checking.
- `29.1.26.txt` — Shell programs covering average of three floating-point numbers, area of a circle, simple interest, largest of three numbers, leap year check, vowel/consonant check, voting eligibility, and job eligibility.
- `5.2.26.txt` — Shell script exercises for printing numbers 1 to N, printing even numbers divisible by 5 between 50 and 100, computing factorials, printing numbers divisible by both 5 and 7 from 50 to 100, and prime number detection.
- `ASSIGNMENT – 3.txt` — WSL shell programs for control-flow practice: print numbers from 1 to N; print even numbers divisible by 5 from 50 to 100; compute factorial; print numbers divisible by 5 and 7; and check prime numbers.
- `Assignment 6` — Python examples demonstrating process behavior with `os.fork()`: parent-child PID printing, `wait()` synchronization, orphan process creation, zombie process creation, and multiple child processes.
- `Assignment 7.py` — Python program for internal and external fragmentation using first-fit and best-fit memory allocation strategies with a sample block and process list.
- `Deadlock Detection Code 1 qstn` — PNG problem statement for the first deadlock detection exercise.
- `Deadlock Detection Code 1(Banker's Algo)` — Python implementation of the Banker's Algorithm that checks whether the system is in a safe state and prints a safe sequence.
- `Deadlock Detection Code 2` — Python deadlock detection example using allocation, request, and available matrices to determine if a deadlock exists.
- `Deadlock Detection Code 2 qstn` — PNG problem statement for the second deadlock detection exercise.
- `FCFS and SJF.txt` — Python script implementing FCFS and SJF scheduling, printing process burst time, waiting time, turnaround time, and average metrics.

## ▶️ How to Compile and Execute Shell Programs

1. Create or open a shell file:

vi program.sh

2. Press i to enter insert mode and write the code.

3. Save and exit:

Esc
:wq

4. Give execute permission:

chmod +x program.sh

5.Run the program:

./program.sh
